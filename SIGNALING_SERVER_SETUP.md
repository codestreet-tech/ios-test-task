# Signaling Server — Setup & Run Guide

Цей сервер використовується як локальний WebRTC signaling server для тестового завдання.

Нижче наведено повні та прості інструкції для запуску.

---

## 1. Встановлення Node.js (якщо ще не встановлений)

На macOS через Homebrew:

```bash
brew install node
```

Перевірити встановлення:

```bash
node -v
npm -v
```

---

## 2. Встановлення залежностей

Перейдіть у папку сервера і виконайте:

```bash
npm install
```

---

## 3. Запуск сервера

У тій самій директорії:

```bash
node index.js
```

Після успішного запуску ви побачите повідомлення:

```
Signaling server listening on http://localhost:3000
```

---

## 4. Адреса сервера

iOS-застосунок повинен підключатися до:

```
http://localhost:3000
```
